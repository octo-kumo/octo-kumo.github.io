It appears that every character is replaced by their MD5 hash.

Solve by pre-computing hash table for every printable character.

import hashlib
import string
import json

table = dict()
for char in string.printable:
    x = hashlib.md5(str(ord(char)).encode()).hexdigest()
    table[int(x)] = char
