The Learning with Errors (LWE) encryption scheme is a lattice-based cryptographic system with the following key mathematical components:

Parameters

  • Modulus: q∈Zq \in \mathbb{Z}q∈Z
  • Dimension: n∈Nn \in \mathbb{N}n∈N
  • Number of samples: N∈NN \in \mathbb{N}N∈N
  • Error distribution standard deviation: σ>0\sigma > 0σ>0

Key Generation

  1. Secret Vector s⃗\vec{s}s: s⃗=[1,t1,t2,…,tn]\vec{s} = \left[1, t_1, t_2, \ldots, t_n\right]s=[1,t1​,t2​,…,tn​] where ti∈[0,q2)t_i \in \left[0, \frac{q}{2}\right)ti​∈[0,2q​)
  2. Public Matrix A∈ZqN×nA \in \mathbb{Z}_q^{N \times n}A∈ZqN×n​: Ai,j∼Uniform(0,q2)A_{i,j} \sim \text{Uniform}\left(0, \frac{q}{2}\right)Ai,j​∼Uniform(0,2q​)
  3. Error Vector e⃗∈ZqN\vec{e} \in \mathbb{Z}_q^Ne∈ZqN​: ei∼Dσe_i \sim \mathcal{D}_{\sigma}ei​∼Dσ​ (Discrete Gaussian Distribution)
  4. Public Parameter PPP: P=[b∣−A]P = \left[b \mid -A\right]P=[b∣−A] where b=(At⃗+e⃗)mod  qb = \left(A \vec{t} + \vec{e}\right) \mod qb=(At+e)modq

Encryption Process

  1. Message Encoding: Mencoded=⌊q2⌋⋅MM_{\text{encoded}} = \left\lfloor\frac{q}{2}\right\rfloor \cdot MMencoded​=⌊2q​⌋⋅M
  2. Ciphertext Computation: C=(PTr⃗+Mencoded)mod  qC = \left(P^T \vec{r} + M_{\text{encoded}}\right) \mod qC=(PTr+Mencoded​)modq where r⃗∈0,1N\vec{r} \in {0,1}^Nr∈0,1N

Decryption Process

Decrypt(C,s⃗)=⌊2q⋅s⃗TC⌋mod  2\text{Decrypt}(C, \vec{s}) = \left\lfloor\frac{2}{q} \cdot \vec{s}^T C\right\rfloor \mod 2Decrypt(C,s)=⌊q2​⋅sTC⌋mod2

Mathematical Derivation of Decryption

The core decryption step can be mathematically expressed as:

s⃗TC≈⌊q2⌋M+small noise\vec{s}^T C \approx \left\lfloor\frac{q}{2}\right\rfloor M + \text{small noise}sTC≈⌊2q​⌋M+small noise

Expanded form: s⃗TC=s⃗T(PTr⃗+Mencoded)\vec{s}^T C = \vec{s}^T \left(P^T \vec{r} + M_{\text{encoded}}\right)sTC=sT(PTr+Mencoded​)

Brute Force Attack Complexity

The attack complexity is bounded by: O(qn)\mathcal{O}\left(q^n\right)O(qn)

With success probability depending on: P(correct recovery)=f(q,n,σ)P\left(\text{correct recovery}\right) = f\left(q, n, \sigma\right)P(correct recovery)=f(q,n,σ)

Key Security Properties

  1. Computational Hardness: LWEs⃗,χ≜Distinguish(A,As⃗+e⃗)\text{LWE}_{\vec{s},\chi} \triangleq \text{Distinguish}\left(A, A\vec{s} + \vec{e}\right)LWEs,χ​≜Distinguish(A,As+e)
  2. Quantum Resistance: Solving LWE is believed to be NP-hard\text{Solving LWE is believed to be } \mathcal{NP}\text{-hard}Solving LWE is believed to be NP-hard

Noise Analysis

The decryption noise follows a distribution: η∼N(0,σ2)\eta \sim \mathcal{N}\left(0, \sigma^2\right)η∼N(0,σ2)

Decryption succeeds when ∣η∣≪q4|\eta| \ll \frac{q}{4}∣η∣≪4q​

Probabilistic Key Recovery

Let H\mathcal{H}H be the hypothesis space of possible secret vectors:

s⃗^=arg⁡max⁡s⃗∈H∣{(Mi,Ci):Decrypt(Ci,s⃗)=Mi}∣\hat{\vec{s}} = \arg\max_{\vec{s} \in \mathcal{H}} \left|\left\{(M_i, C_i) : \text{Decrypt}(C_i, \vec{s}) = M_i\right\}\right|s^=args∈Hmax​∣{(Mi​,Ci​):Decrypt(Ci​,s)=Mi​}∣

def enc(P, M, q):
    N = P.shape[0]
    n = len(M)
    r = np.random.randint(0, 2, (n, N))
    Z = np.zeros((n, P.shape[1]), dtype=np.int32)
    Z[:, 0] = 1
    C = np.zeros((n, P.shape[1]), dtype=np.int32)
    for i in range(n):
        C[i] = (np.dot(P.T, r[i]) + (np.floor(q/2) * Z[i] * M[i])) % q
    return C


def dec(C, s, q):
    M = np.zeros(len(C), dtype=np.int32)
    for i in range(len(C)):
        M[i] = round((np.dot(C[i], s) % q) * (2/q)) % 2
    return M


def crack_n1_lwe(P, q, num_samples=200):
    known_messages = np.random.randint(0, 2, num_samples)
    ciphertexts = enc(P, known_messages, q)
    best_success = 0
    best_ts = []
    for potential_ts in tqdm(range(q**n)):
        potential_ts = np.unravel_index(potential_ts, (q,) * n)
        potential_s = np.concat((np.array([1]), np.array(potential_ts)))
        success_count = 0
        decrypted = dec(ciphertexts, potential_s, q)
        success_count = np.sum(known_messages == decrypted)
        if success_count > best_success:
            best_success = success_count
            best_ts = potential_ts
        if success_count == num_samples:
            break
    recovered_s = np.concat((np.array([1]), np.array(best_ts)))
    success_rate = best_success / num_samples
    return recovered_s, success_rate
